## 📌 Project Overview

This project presents an **interactive Power BI dashboard** built to analyze the **Blinkit grocery sales dataset**.  
The dashboard provides insights into sales performance, product categories, outlet types, and customer ratings to support **data-driven decision making**.

The goal of this project is to demonstrate **data analysis, data visualization, and business intelligence skills** using **Power BI**.

---

## 📊 Dashboard Insights

The dashboard helps analyze:

- 💰 Total Sales Performance  
- 📦 Item Category Distribution  
- ⭐ Customer Ratings  
- 🏬 Outlet Type Performance  
- 📈 Average Sales per Item  
- 🧾 Item Visibility vs Sales Relationship  

These insights help understand **which products perform best and how different outlet types contribute to revenue**.
